SQL GROUP BY Statement

The GROUP BY statement groups rows that have the same values into summary rows â€” for example, to find the number of customers in each country.

It is typically used with aggregate functions like COUNT(), MAX(), MIN(), SUM(), and AVG() to perform operations on each group.

Syntax

SELECT column_name(s) FROM table_name WHERE condition GROUP BY column_name(s) ORDER BY column_name(s);

Example

Count the number of customers in each country:

SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country;

List the number of customers per country, sorted from highest to lowest:

SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country ORDER BY COUNT(CustomerID) DESC;
