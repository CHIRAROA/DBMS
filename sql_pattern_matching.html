SQL LIKE Operator

The LIKE operator is used in a WHERE clause to search for a specified pattern in a column.

There are two wildcards often used with the LIKE operator:

  • The percent sign (%) represents zero, one, or multiple characters.
  • The underscore (_) represents a single character.

LIKE is case-insensitive in some databases.

Syntax

SELECT column1, column2, ... FROM table_name WHERE columnN LIKE pattern;

Examples

Using the % Wildcard

Find values that start with "a":

SELECT * FROM Customers WHERE CustomerName LIKE 'a%';

Find values that end with "a":

SELECT * FROM Customers WHERE CustomerName LIKE '%a';

Find values that contain "or" anywhere:

SELECT * FROM Customers WHERE CustomerName LIKE '%or%';

Using the _ Wildcard

Find values with "r" in the second position:

SELECT * FROM Customers WHERE CustomerName LIKE '_r%';

Find values that start with "a" and are at least 3 characters long:

SELECT * FROM Customers WHERE CustomerName LIKE 'a_%_%';

Find values that start with "a" and end with "o":

SELECT * FROM Customers WHERE CustomerName LIKE 'a%o';

Using NOT with LIKE

Use the NOT operator to exclude a pattern:

Find values that do not start with "a":

SELECT * FROM Customers WHERE CustomerName NOT LIKE 'a%';
