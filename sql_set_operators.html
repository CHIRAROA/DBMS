SQL Table Operators

SQL table operators are used to combine the results of two or more queries. The main table operators are:

  • UNION – combines result sets and removes duplicates.
  • UNION ALL – combines result sets and keeps duplicates.
  • INTERSECT – returns only rows present in both result sets.
  • EXCEPT / MINUS – returns rows from the first result set that are not in the second.

UNION Syntax

SELECT column_list FROM table1 UNION SELECT column_list FROM table2;

UNION ALL Syntax

SELECT column_list FROM table1 UNION ALL SELECT column_list FROM table2;

INTERSECT Syntax

SELECT column_list FROM table1 INTERSECT SELECT column_list FROM table2;

EXCEPT / MINUS Syntax

SELECT column_list FROM table1 EXCEPT SELECT column_list FROM table2;

Note: In Exasol, MINUS is an alias for EXCEPT.

Example

Consider two tables: Employees_US and Employees_EU.

-- Get all unique employees across both regions SELECT name FROM Employees_US UNION SELECT name FROM Employees_EU;
-- Get employees common to both regions SELECT name FROM Employees_US INTERSECT SELECT name FROM Employees_EU;
-- Get employees only in US but not in EU SELECT name FROM Employees_US EXCEPT SELECT name FROM Employees_EU;
