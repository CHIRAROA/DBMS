SQL ORDER BY Keyword

The ORDER BY keyword is used to sort the result set in ascending or descending order.

By default, the ORDER BY clause sorts records in ascending order. To sort records in descending order, use the DESC keyword.

Syntax

SELECT column1, column2, ... FROM table_name ORDER BY column1 [ASC|DESC], column2 [ASC|DESC], ...;

Example: ORDER BY LastName

Select all customers from the "Customers" table, sorted by the LastName column:

SELECT * FROM Customers ORDER BY LastName;

Example: ORDER BY LastName DESC

Select all customers from the "Customers" table, sorted by LastName in descending order:

SELECT * FROM Customers ORDER BY LastName DESC;

Example: ORDER BY Country, CustomerName

Select all customers from the "Customers" table, sorted first by Country, then by CustomerName:

SELECT * FROM Customers ORDER BY Country, CustomerName;
