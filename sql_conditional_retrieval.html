SQL WHERE Clause

The WHERE clause is used to filter records. It helps extract only those records that meet a specific condition.

Syntax

SELECT column1, column2, ... FROM table_name WHERE condition;

Note: The WHERE clause can also be used in UPDATE, DELETE, and other SQL statements.

Example

Select all customers from Mexico:

SELECT * FROM Customers WHERE Country = 'Mexico';

This will return all records from the Customers table where the Country is 'Mexico'.

Text Fields vs. Numeric Fields

In SQL, text values must be enclosed in single quotes. Numeric values should not be enclosed in quotes:

SELECT * FROM Customers WHERE CustomerID = 1;

Ensure you use the appropriate syntax to avoid errors and retrieve accurate results.
