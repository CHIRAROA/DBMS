SQL String Functions

SQL offers a variety of string functions to manipulate text data efficiently. These functions allow developers to extract, modify, combine, and analyze strings in powerful ways.

Common SQL String Functions

Function Description Example
ASCII() Returns the ASCII value of the first character in a string.
SELECT ASCII('A') AS AsciiValue;
CHAR() Returns the character corresponding to a given ASCII code.
SELECT CHAR(65) AS Character;
CHARINDEX() Finds the position of a substring within a string.
SELECT CHARINDEX('world', 'Hello world') AS Position;
CONCAT() Joins two or more strings together.
SELECT CONCAT('Hello', ' ', 'World') AS Greeting;
LEFT() Returns the first N characters from the left of a string.
SELECT LEFT('Hello World', 5) AS LeftPart;
RIGHT() Returns the last N characters from the right of a string.
SELECT RIGHT('Hello World', 5) AS RightPart;
LEN() Returns the number of characters in a string.
SELECT LEN('Hello World') AS Length;
LOWER() Converts all letters in a string to lowercase.
SELECT LOWER('HELLO WORLD') AS Lowercase;
UPPER() Converts all letters in a string to uppercase.
SELECT UPPER('hello world') AS Uppercase;
LTRIM() Removes leading spaces from a string.
SELECT LTRIM(' Hello World') AS Trimmed;
RTRIM() Removes trailing spaces from a string.
SELECT RTRIM('Hello World ') AS Trimmed;
REPLACE() Replaces all occurrences of a substring with another substring.
SELECT REPLACE('Hello World', 'World', 'SQL') AS Replaced;
REVERSE() Reverses the characters in a string.
SELECT REVERSE('Hello') AS Reversed;
STUFF() Deletes part of a string and inserts another string at the specified position.
SELECT STUFF('Hello World', 6, 5, 'SQL') AS Stuffed;
SUBSTRING() Extracts part of a string starting at a given position and length.
SELECT SUBSTRING('Hello World', 7, 5) AS Extracted;

Conclusion

String functions are essential for text manipulation in SQL. They allow precise control over how string data is handled, cleaned, and presented in your queries.
