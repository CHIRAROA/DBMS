SQL CREATE TABLE Statement

The CREATE TABLE statement is used to create a new table in a database.

Syntax

CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, ... );

Example

This example creates a table named Customers with columns for CustomerID, CustomerName, ContactName, Address, City, PostalCode, and Country:

CREATE TABLE Customers ( CustomerID int, CustomerName varchar(255), ContactName varchar(255), Address varchar(255), City varchar(255), PostalCode varchar(255), Country varchar(255) );

Each column in the table is defined with a name and a data type.
