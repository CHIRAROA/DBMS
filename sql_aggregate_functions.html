SQL Aggregate Functions

An aggregate function performs a calculation on a set of values and returns a single value. They're commonly used with the GROUP BY clause to group result sets.

Common SQL aggregate functions include:

  • MIN() – Returns the smallest value in a set.
  • MAX() – Returns the largest value in a set.
  • COUNT() – Returns the number of rows that match a condition.
  • SUM() – Returns the total sum of a numeric column.
  • AVG() – Returns the average value of a numeric column.

Note: All aggregate functions ignore NULL values, except COUNT().

COUNT() Function

Returns the number of rows that match a specified condition.

Syntax

SELECT COUNT(column_name) FROM table_name WHERE condition;

Example

Count the number of products:

SELECT COUNT(ProductID) FROM Products;

AVG() Function

Returns the average value of a numeric column.

Syntax

SELECT AVG(column_name) FROM table_name WHERE condition;

Example

Average price of all products:

SELECT AVG(Price) FROM Products;

SUM() Function

Returns the total sum of a numeric column.

Syntax

SELECT SUM(column_name) FROM table_name WHERE condition;

Example

Total quantity in the OrderDetails table:

SELECT SUM(Quantity) FROM OrderDetails;

Total quantity for ProductID 11:

SELECT SUM(Quantity) FROM OrderDetails WHERE ProductID = 11;

MIN() and MAX() Functions

MIN() returns the smallest value in a column, and MAX() returns the largest.

Syntax

MIN:

SELECT MIN(column_name) FROM table_name WHERE condition;

MAX:

SELECT MAX(column_name) FROM table_name WHERE condition;

Example

Find the lowest and highest prices:

MIN:

SELECT MIN(Price) FROM Products;

MAX:

SELECT MAX(Price) FROM Products;
